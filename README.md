# Rust_Bevy_tutorial
Я зробив гру м'яч на Rust з Bevy. Мій проект назвивається Rust_Bevy_tutorial, тому що я вивчав теорію про Bevy на Rust. Тут в мене багато файлів.
## Що за гра м'яч?
Гра з м'ячем – це вид спорту або розвага, де використовується м'яч, а метою є його кидання, биття, ловлення або забивання в ціль (ворота), що розвиває спритність, координацію та реакцію, включаючи такі ігри як футбол, баскетбол, волейбол, а також прості дитячі ігри на кшталт "Їстівне-неїстівне".
Основні елементи:  
-М'яч: Сферичний предмет, що може бути різних розмірів та матеріалів.  
-Мета: Забити м'яч у ворота, перекинути його через сітку, влучити в ціль або просто не дати впасти.  
-Інструменти: Можуть використовуватися руки, ноги, бити (як у крикеті).  
Типи ігор:  
-Командні спортивні ігри: Футбол (ногами), баскетбол (руками, ведення), волейбол (руками, через сітку), водне поло (у воді).  
Дитячі рухливі ігри:  
-"Їстівне-неїстівне": Ведучий кидає м'яч і називає предмет; якщо їстівний - ловити, якщо ні - відбивати.  
-"Гонка м'ячів": Гравці швидко передають м'яч по колу.  
-Ігри на координацію: Підкидання і ловіння м'яча, перенесення його між тілом (наприклад, між колінами).  
## Що таке Rust?
Rust — це сучасна багатопарадигмова мова програмування, що фокусується на швидкості, безпеці пам'яті та паралелізмі, розроблена Mozilla Research як системна мова з продуктивністю на рівні C++, але без багатьох її проблем, що робить її ідеальною для ОС, веб-серверів, вбудованих систем, ігор, блокчейну та інших високопродуктивних застосунків. 
Де використовується:  
-Операційні системи та ядра  
-Веб-розробка (Backend, WebAssembly)  
-Ігрова індустрія  
-Блокчейн та Web3  
-Високонавантажені сервіси (Twitter, Dropbox, AWS)   
Важливо:  
-Існує також відеогра "Rust" - це симулятор виживання від першої особи, що є окремим проєктом, не пов'язаним з мовою програмування.  
## Що таке Bevy?
Bevy — це сучасний, модульний і швидкий ігровий рушій (game engine) та фреймворк для розробки застосунків, написаний на мові програмування Rust, відомий своїм підходом, орієнтованим на дані (Data-Driven Design) та використання Архітектури Сутність-Компонент-Система (ECS). Він надає розробникам простий, але потужний інструмент для створення ігор та програм, фокусуючись на продуктивності та зручності використання.  
Ключові характеристики Bevy:  
-Rust-based: Повністю побудований на мові Rust, що забезпечує безпеку пам'яті та високу швидкість.  
-ECS-орієнтований: Використовує архітектуру Entity Component System (ECS), яка є ефективною для управління складними ігровими даними та логікою.  
-Модульний: Дозволяє додавати лише необхідні компоненти, роблячи його легким та гнучким.  
-Простий та зручний: Спрямований на те, щоб бути "освіжаючим" і простим у використанні, знижуючи поріг входу для розробників.  
-Відкритий код (Open-Source): Доступний для всіх, що сприяє його розвитку спільнотою.
